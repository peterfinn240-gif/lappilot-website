2. Erhobene Daten

Wir erheben folgende Daten zur Nutzung der App:

3. Zweck der Datenverarbeitung

Die Daten werden ausschliesslich verwendet für:

  • Bereitstellung der App-Funktionen

  • KI-gestützte Generierung von Lernmaterialien

4. Datenweitergabe

Deine Daten werden nicht an Dritte verkauft. Eine Weitergabe erfolgt nur an:

  • Supabase (Datenbankdienst, Server in der EU)
  • Apple/Google (In-App-Käufe, gemäss deren Datenschutzrichtlinien)
  • OpenAI/Anthropic (KI-Verarbeitung, ohne personenbezogene Daten)

5. Speicherung und Sicherheit

Deine Daten werden auf Servern in der EU gespeichert (Supabase). Die Übertragung erfolgt verschlüsselt (TLS). Hochgeladene Dokumente werden nach der KI-Verarbeitung automatisch gelöscht.

Kontaktiere uns unter info@lappilot.ch für alle Anfragen zum Datenschutz.
